# 防彦美老年痴呆训练

一款包含两个益智小游戏的网页应用，专为锻炼大脑设计。

## 游戏介绍

### 1. 自由拼图
- 上传自己喜欢的图片
- 选择不同难度级别（16片、32片、64片、128片）
- 点击查看原图按钮可以预览原图
- 计时功能记录完成时间

### 2. 数字消消乐
- 5x5的游戏网格
- 数字范围从-2到4
- 3个或以上相同数字连成一片即可消除
- 使用加减符号道具调整数字
- 道具栏每次使用后会随机刷新

## 如何在GitHub Pages上游玩

### 方法一：直接使用GitHub Pages

1. 将此仓库克隆或下载到本地
2. 创建一个新的GitHub仓库
3. 将所有文件上传到GitHub仓库
4. 在仓库设置中开启GitHub Pages：
   - 进入仓库的 **Settings**
   - 找到 **Pages** 选项
   - 在 **Source** 中选择 `main` 分支（或你使用的分支）
   - 点击 **Save**
5. 等待几分钟，GitHub Pages会生成一个网址
6. 通过生成的网址即可游玩

### 方法二：本地游玩

直接用浏览器打开 `index.html` 文件即可开始游玩。

## 游戏规则

### 自由拼图
1. 点击上传图片区域选择一张图片
2. 选择拼图分片数量
3. 点击开始拼图
4. 点击两块拼图进行交换
5. 完成拼图后会显示用时

### 数字消消乐
1. 观察游戏板上的数字
2. 点击可消除的石头（3个及以上相同数字连成一片）
3. 或拖动道具栏的加减符号到石头上改变数字
4. 每次使用道具后，道具栏会随机刷新
5. 尽可能获得高分！
